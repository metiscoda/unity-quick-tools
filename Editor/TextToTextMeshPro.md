# Text to TextMeshPro

## Setup

Project Settings > TextMeshPro > Settings > Default Font Asset (Set to the TMP font you want to use)

## Usage

1) Select Text(s) in Hierarchy 
1) Select: Tools > Text To TextMeshPro
